4. Emergency: Token/Auth Issues

üî¥ Forge Goes Silent (Usage Cap Hit)

Symptom: Messages sent but no reply for 30+ minutes

Cause: Monthly usage allowance exhausted

Fix: Wait for reset (usually next day/period)
Or: Switch to Sonnet model temporarily
   openclaw config set model anthropic/claude-sonnet-4-6

üî¥ Gateway Crash Loop (OAuth 403)

Symptom: Gateway restarts every 10 seconds, errors in logs

Cause: Auth error misclassified as rate limit ‚Üí cooldown loop

Fix:
1. Edit ~/.openclaw/auth-profiles.json
2. Set "errorCount": 0
3. Remove "cooldownUntil" field
4. openclaw gateway restart

üî¥ GitHub PAT Expired

Symptom: Git push fails with 403

Fix:
1. Go to github.com ‚Üí Settings ‚Üí Developer Settings ‚Üí PATs
2. Generate new token (repo + workflow scopes)
3. git remote set-url origin https://NEW_TOKEN@github.com/AntonUrso2811/thalyve-tools.git
4. Same for forge-workspace repo

üî¥ Account Banned (Worst Case)

Symptom: All API calls return 403/banned

Recovery:
1. Contact Anthropic support immediately
2. Switch to Layer 3 emergency token (if set up)
3. Or: Set up new account + new subscription
4. Update openclaw.json with new OAT token
5. openclaw gateway restart

6. Cost Optimization

Current Monthly Cost

Claude Max subscription: ~$100-200/month (covers Opus + Sonnet)

No API costs. No surprise bills.

How We Save

  • Dynamic routing: Opus for quality tasks, Sonnet for mechanical ones
  • Memory files > chat context: Save to files, not long conversations
  • Compaction awareness: Wrap up before context fills, save state
  • Cron on Sonnet: All automated jobs run on cheaper model
  • No API fallback: Subscription only ‚Äî predictable costs

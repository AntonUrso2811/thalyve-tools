üî¥ Problem 1: Forge stops responding on Telegram

1

Check if Forge is alive

Open Safari on the Mac ‚Üí go to http://127.0.0.1:18789

If the webchat loads and Forge responds ‚Üí he's alive, it's just a Telegram routing issue.

2

Restart the gateway

Wait 60 seconds. Message Forge on Telegram.

3

Close any webchat tabs

If you have http://127.0.0.1:18789 open in a browser, close that tab. The webchat can hijack Telegram routing. This happened to us on Feb 25.

4

If still broken ‚Äî stop and start

openclaw gateway stop
# Wait 10 seconds
openclaw gateway start
5

Check the logs

openclaw logs --follow

Look for error messages. Screenshot them and send to OpenClaw Discord.

üü° Problem 2: Rate limit / API cooldown

What it looks like: Forge responds slowly, gives errors, or goes silent.

What to do:

  1. WAIT. Rate limits reset automatically (5-15 minutes)
  2. Check your Anthropic dashboard: console.anthropic.com ‚Üí Usage
  3. If you've hit a spend limit, increase it in the dashboard under Settings ‚Üí Limits
  4. After waiting: openclaw gateway restart


What NOT to do: ‚ùå Don't reinstall. ‚ùå Don't delete files. ‚ùå Don't panic. Just wait.

üåç VPS Setup ‚Äî Travel Without Worrying

Right now, Forge runs on the Mac at home. If you close the lid, turn it off, or leave it behind ‚Äî Forge goes offline. A VPS fixes this permanently.

What is a VPS?

A Virtual Private Server ‚Äî a computer in the cloud that's always on, always connected. Think of it as a Mac that never sleeps, never loses WiFi, and runs 24/7 from a data centre.

VPS Setup ‚Äî Step by Step

1

Create a VPS

Pick a provider above. Choose Ubuntu 24.04, minimum 2GB RAM, 1 vCPU. Location: London (closest to you).

2

SSH into the VPS

# You'll get an IP address and password from the provider
ssh root@YOUR_VPS_IP
3

Install Node.js + OpenClaw

# Install Node.js
curl -fsSL https://deb.nodesource.com/setup_22.x | bash -
apt-get install -y nodejs

# Install OpenClaw
npm install -g openclaw

# Set up OpenClaw
openclaw init
4

Restore Forge's workspace from GitHub

cd ~/.openclaw/workspace
git init
git remote add origin https://github.com/AntonUrso2811/forge-workspace.git
git fetch origin
git reset --hard origin/main
5

Set up your Anthropic API key

openclaw agents add main
# Follow the prompts ‚Äî enter your API key from console.anthropic.com
6

Connect Telegram

openclaw channels add telegram
# Follow the prompts ‚Äî enter your Telegram bot token
7

Start the gateway

openclaw gateway start

Forge is now running on the VPS. 24/7, anywhere in the world. Close your laptop, go travelling, it keeps running.

When to move to VPS

My recommendation:

Not yet. Your Mac setup is fine for now. Move to a VPS when:

  • You're about to travel and need Forge online while you're away
  • You're tired of keeping the Mac open and connected
  • You want true 24/7 reliability

When you're ready, tell Forge and he'll do the migration for you. All files are on GitHub ‚Äî the move takes ~15 minutes.

What about the Mac after VPS?

You can either:

  • Keep both running ‚Äî Mac as development, VPS as production (advanced)
  • Switch fully to VPS ‚Äî shut down OpenClaw on the Mac, use it as a normal laptop
  • Use Mac as backup ‚Äî if VPS goes down, start gateway on Mac as emergency fallback

🔱 Forge Quick Reference

How your AI Chief of Staff works — and how to keep it running at full speed

🏗️ How Forge Works

Three layers, all running on your Mac Mini:

1
Gateway
Daemon service on the Mac. Routes messages between Telegram ↔ Forge. Always running via LaunchAgent.
2
Brain (Claude Opus)
API calls to Anthropic's servers. Processes every message, makes decisions, writes code, generates strategy.
3
Tools
Browser (Higgsfield, web), file system, shell commands, Shopify API, GitHub, Telegram — everything Forge uses to execute.

🛠️ Terminal Commands

Open Terminal on the Mac and run these. Tap any command to copy.

90% Fix — Restart Gateway

openclaw gateway restart

Fixes most issues — browser timeouts, message delivery, stuck responses.

If Restart Doesn't Work

openclaw gateway stop
openclaw gateway start

Check Status

openclaw gateway status
openclaw status

Send the output to Forge for diagnosis.

Nuclear Option — Full Reset

openclaw gateway stop && killall node && openclaw gateway start

Only use if everything else fails. Kills all Node processes and restarts clean.

🚀 How To Keep Forge Fast

  • One task at a time — Forge is fastest when focused on a single deliverable
  • Voice notes work great — keep doing what you're doing
  • Don't repeat context — Forge remembers everything within a session
  • Long sessions slow down — after 50+ messages, responses get slower (more context). This is normal. Gateway restart won't fix it.
  • Browser tasks are slowest — Higgsfield/web automation = 10-30s per action. API calls (Shopify, GitHub) = instant
  • Sub-agents for heavy lifting — parallel work without slowing the main chat

💬 Quick Prompts

  • "Deep Work" — Forge gives task briefing and goes silent
  • "Status" — What's currently in progress
  • "Restart" — Forge restarts the gateway itself
  • Plain English / voice notes — no special format needed

⚠️ The One Thing You Should Never Do

Don't close the Terminal window running the gateway.

If the Mac restarts, the gateway auto-starts via LaunchAgent. But if you manually kill it, run openclaw gateway start.

🚀 Product Launch Pipeline

From Greenlight to LIVE — v1.0 | 2 March 2026

Anton says "This is the product" → Forge delivers a LIVE, selling product page.

No questions. No check-ins. Just delivery. 🔱

The Pipeline — 10 Steps

1
Brand Kit
⏱ 30 mins
🔧 Claude (Forge)
Target market psychology, colour palette (hex codes), typography, photography style, brand voice, compliance language. Every asset that follows must match this kit.
📄 reference/brand-kits/[handle]-brand-kit.md
2
Product Copy
⏱ 45 mins
🔧 Claude (Forge) + BBA/ESS Frameworks
Hero headline, Problem→Agitation→Solution, benefits (6 cards), how-to (3 steps), comparison table, pain points, purity claims, FAQ (7-10 Qs), CTAs, SEO meta.
📄 copy/[handle]-pdp-copy.md
3
Product Photography
⏱ 1-2 hrs
🔧 Higgsfield (Nano Banana Pro)
ALWAYS attach packaging-MASTER reference. One prompt per gen, batch 4/4, vision-score every batch (8/10+ only). Gallery: Hero → Ritual → Ingredients → Comparison → Trust.
📁 reference/brand-kits/ads/[handle]/
4
Selling Infographics
⏱ 1 hr
🔧 Custom HTML + Chrome Headless Render
Text-heavy selling images AI can't generate: benefits grid, how-to steps, comparison chart, pain points, purity claims. All 1080×1080, brand kit colours/fonts.
📁 templates/gallery-infographics/renders/
5
Custom Liquid Sections
⏱ 1 hr
🔧 Claude (Forge)
Full custom HTML/CSS Liquid sections: comparison, benefits grid, how-to, purity, pain points, FAQ accordion. Mobile responsive, brand colours, hover effects.
📁 shopify/sections/[handle]-*.liquid
6
Deploy Sections to Shopify
⏱ 15 mins
🔧 Shopify Admin API
Upload Liquid sections → Create product-specific template (product.[handle].json). Each product gets its OWN template file — visible in Shopify dashboard.
📄 templates/product.[handle].json on Shrine theme
7
Upload Images (Selling Order)
⏱ 15 mins
🔧 Shopify Admin API
Upload gallery images in EXACT selling order. Compress to JPEG (200-500KB). Slot 1 = hero (this is the thumbnail everywhere).
Product gallery on Shopify
8
Assign Template to Product
⏱ 5 mins
🔧 Shopify Admin API
Set template_suffix on product to use the custom template. Product now renders with all custom sections.
Product uses product.[handle].json
9
QC Check
⏱ 30 mins
🔧 Browser + Vision Analysis
All images load in order, all sections render (mobile + desktop), brand colours match, ATC works, page speed OK, SEO meta set, no dead content.
QC pass/fail report
10
Deliver to Anton
⏱ 5 mins
🔧 Telegram
Live URL, desktop + mobile screenshots, gallery lineup. "It's live. Check it."
📱 Telegram message with link + screenshots
⏱ Total: ~5-6 hours from greenlight to LIVE
